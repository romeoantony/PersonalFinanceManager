# Personal Finance Manager

A comprehensive Personal Finance Manager application built with WPF and .NET, designed to help users track expenses, manage categories, and visualize financial data.

## Features

- **Expense Tracking**: Add, edit, and delete expenses with ease.
- **Category Management**: Organize expenses into custom categories.
- **Visual Analytics**: View interactive charts and graphs using LiveCharts2 to understand spending habits.
- **Monthly Summaries**: Generate detailed summaries of monthly expenses.
- **Data Export**: Export financial data to PDF and Excel formats for external analysis.
- **Local Database**: Securely stores data locally using SQLite.

## Technologies Used

- **WPF (Windows Presentation Foundation)**: For the desktop user interface.
- **.NET**: Core framework.
- **SQLite**: Lightweight, serverless database engine.
- **LiveCharts2**: For beautiful and interactive data visualization.
- **CommunityToolkit.Mvvm**: For implementing the Model-View-ViewModel pattern.
- **QuestPDF**: For generating PDF reports.

## Getting Started

### Prerequisites

- .NET SDK (compatible with the project version)
- Visual Studio or a compatible IDE

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/romeoantony/Personal-Finance-Manager.git
    ```
2.  Navigate to the project directory.
3.  Open the solution file (`.sln`) in Visual Studio.
4.  Restore NuGet packages.
5.  Build and run the application.

## Usage

- Launch the application.
- Navigate through the tabs to access different features.
- Use the "Add Expense" form to record new transactions.
- Check the "Dashboard" for visual insights.
